# 🌧️ Machine Learning Group Assignment  
## Extreme Rainfall Risk Classification for Sri Lankan Cities

## 📌 Project Overview
This project focuses on the evaluation and comparison of classical machine learning models to classify **extreme rainfall risk levels** in Sri Lankan cities using a real-world weather dataset.

The system applies **supervised machine learning classification techniques** to analyze weather patterns and predict rainfall severity levels.

The main objective is to support **early risk identification and decision-making** by providing data-driven rainfall classification using historical meteorological factors.

---

## 🎯 Objectives
- Analyze Sri Lankan weather data for rainfall risk prediction  
- Preprocess and prepare the dataset for machine learning  
- Train and compare multiple classification models  
- Identify the best-performing model based on evaluation metrics  
- Predict rainfall severity levels accurately  

---

## 🧠 Machine Learning Models Used
The following models were implemented and compared:

- Logistic Regression  
- Decision Tree  
- Random Forest  
- Support Vector Machine (SVM)

---

## 📊 Dataset
**Dataset:** Sri Lanka Weather Dataset  
🔗 Kaggle Dataset:  
https://www.kaggle.com/datasets/rasulmah/sri-lanka-weather-dataset

Features include:
- Temperature  
- Humidity  
- Wind Speed  
- Atmospheric Pressure  
- City Information  
- Rainfall Measurements  

### Rainfall Classification Categories
- No Rain  
- Light Rain  
- Moderate Rain  
- Heavy Rain
